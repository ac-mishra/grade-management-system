# Grade Management System

## Project Overview

Grade Management System is a console-based Java application developed using Core Java concepts.

The system allows users to:
- Add student marks
- View all student records
- Calculate averages
- Assign grades
- Find top performers
- Generate performance reports

This project demonstrates Java concepts such as Arrays, Loops, ArrayList, Exception Handling, Validation, and Object-Oriented Programming.
